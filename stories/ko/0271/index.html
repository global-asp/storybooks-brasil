둘 Dois Dos

Carole Bloch

Richard MacIntosh

Jong Yong Park

Scarlet Kim

Korean

Level 1

잡을 수 있는 두 작은 손.

Duas mãozinhas para segurar.

Dos pequeñas manos para sostener.

찰 수 있는 두 작은 발.

Dois pezinhos para chutar.

Dos pequeños pies para patear.

볼 수 있는 두 작은 눈.

Dois olhinhos para ver.

Dos pequeños ojos para ver.

들을 수 있는 두 작은 귀.

Duas orelhinhas para ouvir.

Dos pequeños oídos para escuchar.

그리고 안을 수 있는 두 작은 팔!

Dois braços carinhosos para ABRAÇAR!

¡Y dos brazos amorosos para ABRAZAR!

Written by: Carole Bloch
Illustrated by: Richard MacIntosh
Translated by: Jong Yong Park
Read by: Scarlet Kim
Language: Korean
Level 1
