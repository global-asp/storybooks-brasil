학교 옷 Uniformes da escola Ropa de escuela

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Jong Yong Park

Scarlet Kim

Korean

Level 1

이 드레스는 너무 길어.

Este vestido é longo.

Este vestido es largo.

이 겉옷은 커.

Esta camisa é grande.

Este suéter es grande.

이 가방은 커.

Esta bolsa é grande.

Este bolso es grande.

이 벨트는…

Este cinto é...

Este cinturón es...

이 모자는 작아.

Este chapéu é pequeno.

Este sombrero es pequeño.

양말은 짧아.

As meias são curtas.

Los calcetines son cortos.

하지만 이 신발은 새 거야.

Mas estes sapatos são novos.

Pero estos zapatos son nuevos.

…잘 맞네.

...eles servem muito bem.

... Me quedan bien.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini, Ingrid Schechter
Translated by: Jong Yong Park
Read by: Scarlet Kim
Language: Korean
Level 1
